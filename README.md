# GlowGuide  

Clients can create a personal user account.

Clients can log in using passwords.

Clients can upload images of their skin concerns.

Clients can receive personalized skincare product recommendations.

Clients can add skincare products to a shopping cart.

Clients can purchase skincare products online.

Clients can view their previous purchase history.

Specialists can create a professional user account.

Specialists can upload files or images of their professional certifications.

Specialists can view messages and photos submitted by clients.

Specialists can reply to client messages with personalized advice.
